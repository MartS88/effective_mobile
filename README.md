
## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.



## Installation

```bash
cd effective_mobile
```

```bash
$ npm install
```

```bash
npx sequelize-cli db:migrate
```


## Running the app



```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash

# e2e tests

# test coverage
```
